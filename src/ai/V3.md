# V3 — Determinized Alpha-Beta Search

> 源码：`src/ai/V3-search.js`

## 算法

```
Determinizated Alpha-Beta Search
  ├── 对未知牌采样 N 个完备信息世界
  ├── 每个世界中运行 Alpha-Beta 搜索
  ├── 投票聚合 → 选票数最多的走法
  └── 超时保护 → 确保 ≤ timeLimitMs
```

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `sampleCount` | 100 | 采样世界数 |
| `depth` | 6 | 搜索深度（ply） |
| `timeLimitMs` | 1000 | 每步时间上限 |
| `endgameDepth` | 8 | 残局搜索深度 |
| `endgameThreshold` | 5 | 剩余 ≤ 5 轮进入残局模式 |

## 评估函数因子

| 因子 | 权重 | 说明 |
|------|------|------|
| 得分数差 | ×100 | 最核心指标 |
| 天龙人分牌差 | ×30 | 最危险牌型 |
| 大王小王差 | ×25 | 最强特权 |
| 分数牌持有差 | ×20 | 潜在得分 |
| 特权牌差 | ×15 | 花色控制 |
| 桌面分牌差 | ×10 | 即将出动的分 |
| 手牌数差 | ×5 | 资源量 |
| 桌面牌数差 | ×3 | 可出牌数 |
| 残局放大 | ×1.5 | 领先放大 |
| 末期手牌差 | ×3 | 消耗评估 |

## 用法

```js
// 对战
const result = runHeadToHead('V1', 'V3', 50);

// 浏览器观察参数调优
V3_CONFIG.sampleCount = 50;   // 降采样提速
V3_CONFIG.depth = 4;           // 降低搜索深度
```
