# 简单策略 (easy)

> 源码：`src/ai/easyBot.js`

## 策略概述

最简单的启发式策略。决策只用当前牌面信息，不做推断。

## 跟牌规则（对方先出）

1. **对方出分牌，能赢就压** → 出的赢牌中选最小的分数牌
2. **自己分数牌能赢** → 出最小的赢牌
3. **默认** → 出最小的非分数牌

## 先手规则（自己先出）

1. **优先出天龙人花色分牌** → 最安全，出最小的
2. **只剩分数牌** → 出最小的
3. **默认** → 出最小的非分数牌

## 特点

- 不做任何牌面记忆或推断
- 不窥探对手手牌
- 没有弃牌策略——赢不了时直接出最小牌
- 没有残局意识——不区分早期和末期
- 没有桌面牌偏好——不区分手牌和桌面明牌
