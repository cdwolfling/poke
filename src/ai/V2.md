# 普通策略 (normal)

> 源码：`src/ai/normalBot.js`

## 策略概述

改进策略，包含牌面记忆、桌面牌优先、策略性弃牌、残局激进、花色空缺检测等。

## 跟牌规则（对方先出）

### ① 对方出分牌 + 能赢
```
→ 出最小的能赢的分数牌（桌面牌优先）
```

### ② 自己的分数牌能赢
```
→ 出最小的能赢的分数牌（桌面牌优先）
```

### ③ 赢不了 → 策略性弃牌
```
→ 优先弃同花色的非分数牌（保留分数牌）
→ 否则弃同花色最小牌
→ 无同花色则全局最小牌垫底
```

## 先手规则（自己先出）

### ④ 天龙人花色分牌（最安全）
```
→ 出最小的天龙人分数牌
```

### ⑤ 残局进攻（最后5轮）
```
→ 先出特权分牌
→ 最后2轮无特权也出分数牌
```

### ⑥ 花色空缺检测
```
对手某花色已全部出现在已出牌中 → 出该花色高牌（J/Q/K/A）
```

### ⑦ 只剩分数牌
```
→ 出最小分数牌
```

### ⑧ 默认
```
→ 出最小非分数牌（桌面牌优先）
```

## 改进点

| 改进 | 作用 |
|------|------|
| 桌面牌优先 | 同 rank+suit 时优先出桌面明牌，隐藏手牌 |
| 策略性弃牌 | 跟牌赢不了时优先弃同花色非分牌，保留大牌和分牌 |
| 残局激进 | 最后5轮更积极出分；最后2轮无特权也出 |
| 计牌 | 追踪已出5/10/K张数，评估剩余牌力 |
| 花色空缺 | 检测对手某花色已空，出该花色高牌抢分 |

## 数据结构

```
seenKeys: Set<string>        # 所有已知牌的 key（rank+suit）
unknownCards: Card[]          # 从54张中扣除已知牌
remainingScore: {5,10,K}     # 剩余分数牌计数
```
